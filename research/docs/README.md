# Research Docs

这里放跨报告的研究文档、研究日志与提交材料。

## 核心文件
- `research/docs/RESEARCH_SUMMARY.md`: 单文件研究总览，给人或模型快速上手。
- `research/docs/research_log/`: 阶段日志、迁移基线、归档记录。
- `research/docs/agentreport.md`: 历史过程记录。
- `research/docs/submission/`: 提交材料 PDF。

## 常用维护命令
- `python3 scripts/update_research_summary.py`
- `python3 scripts/check_docs_paths.py`
- `python3 scripts/check_legacy_usage.py`
- `python3 scripts/cleanup_local.py`

## 与其他区域的关系
- 报告正文和资产在 `research/reports/`
- 历史运行归档在 `research/archives/`
- 网站、schema、自动化能力在 `platform/`
