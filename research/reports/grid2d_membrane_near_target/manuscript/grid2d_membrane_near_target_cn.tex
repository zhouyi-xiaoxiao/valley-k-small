\section*{结论先行}
本报告在二维长方形域上新增两条研究线，并且与旧报告的 phase 判据、峰谷图式、分解解释保持一致。
\begin{enumerate}
  \item \textbf{有 corridor、单 target：}将原全反射走廊墙替换为半透膜，分别研究对称膜（$\kappa_{\mathrm{c\to o}}=\kappa_{\mathrm{o\to c}}$）与方向不对称膜（$\kappa_{\mathrm{c\to o}}\neq\kappa_{\mathrm{o\to c}}$）。
  \item \textbf{无 corridor、双 target：}将一个 target 放在起点附近，另一个保持远端，研究“近目标快峰 + 远目标慢峰”是否稳定形成双峰。
\end{enumerate}

核心结果：
\begin{itemize}
  \item 对称半透膜扫描中，clear-double 在超低泄漏区形成连续带（见图~\ref{fig:sym-phase} 与表~\ref{tab:sym-w16}）；随后进入 weak-double 过渡带，再增大渗透率后转为单峰。
  \item 两个方向不对称代表案例都保持 clear-double：易出难回 $(\kappa_{\mathrm{c\to o}},\kappa_{\mathrm{o\to c}})=(0.002,0)$ 与难出易回 $(0,0.002)$。新的 exact 空间占据图与方向通量图直接说明了 peak1 和 peak2 分别由哪些空间轨迹贡献。
  \item 无 corridor 双 target 现在分成两个锚点来讲：physics anchor $(b_x,d,\Delta y)=(0.12,2,2)$ 给出 near/far 质量约 $0.324/0.676$，三处特征时刻约为 $(t_{p1},t_v,t_{p2})=(7,196,469)$；专用 clear-instance（showcase 最优）仍为 $(0.12,4,1)$，near/far 质量约 $0.350/0.650$，且 valley/max 约 $0.007$。
\end{itemize}

\section{模型与参数}
\subsection{统一基线}
沿用已有二维长方形框架：
\[
L_x=60,\quad q=0.8,\quad x_s=7,\quad x_t=58,
\]
一阶邻域随机游走 + 惰性停留；边界为反射边界。
单 target corridor 配置使用本轮优化后的锚点：$h=2$、$m=5$、$\delta_{\rm core}=1.00$、$\delta_{\rm open}=0.55$、$b_x=-0.08$。

\subsection{半透膜定义}
原走廊墙是全反射（穿越概率 0）。本报告改为半透膜：
\[
P(\text{穿越膜边})=\kappa,\qquad P(\text{被反射})=1-\kappa.
\]
本轮主线的不对称是\emph{方向不对称}：对同一条膜边，corridor$\to$outer 与 outer$\to$corridor 两个方向允许不同的通过率，
\[
\kappa_{\mathrm{c\to o}},\qquad \kappa_{\mathrm{o\to c}}.
\]
对称膜：$\kappa_{\mathrm{c\to o}}=\kappa_{\mathrm{o\to c}}=\kappa$；
方向不对称膜：$\kappa_{\mathrm{c\to o}}\neq\kappa_{\mathrm{o\to c}}$。

\subsection{分类框架}
保持“峰-谷-峰”窗口分析，并加入新的 4 类标签：
\begin{itemize}
  \item $L$: 是否发生膜泄漏（是否穿过膜边）；
  \item $R$: 是否发生 committor 承诺后回撤（recross）。
\end{itemize}
故有四类：\texttt{L0R0}, \texttt{L0R1}, \texttt{L1R0}, \texttt{L1R1}。

\section{结果 B：无 corridor + 双 target（近起点目标）}
本节移除 corridor，仅保留反射边界与全局 $b_x$，并把一个 target 放到起点附近。Track B 现在明确分成两个角色：\emph{physics anchor} $(b_x,d,\Delta y)=(0.12,2,2)$ 用来解释 second peak 的物理来源，\emph{showcase clear-instance} $(0.12,4,1)$ 用来给出最清晰的双峰示意。旧的 $(d,b_x)$ 粗扫描仍然保留作为背景，但正文顺序改成“配置 $\rightarrow$ 分支分流 $\rightarrow$ continuation $\rightarrow$ loss mode”，避免还没看清 second peak 到底消失了什么，就先做相图分类。

\section{方法参考（committor + 2×2 分类）}
\subsection*{phase/sep-score 的可复现定义}
为保证相图可复现，本文直接采用代码中的判据（与 \path{packages/vkcore/src/vkcore/grid2d/rect_bimodality/cli.py} 一致）：
\paragraph{(1) 双峰探测（\texttt{find\_two\_peaks}）}
令 $\tilde f=\mathrm{smooth}_7(f)$（窗口 7 平滑）。候选峰点需满足局部极大且高于相对阈值
\[
\tilde f(t)>\tilde f(t-1),\quad \tilde f(t)\ge \tilde f(t+1),\quad \tilde f(t)\ge 0.06\,\max \tilde f.
\]
从候选集合中选两峰 $(t_{p1},t_{p2})$ 时，额外要求（代码默认值）：
\[
t_{p2}-t_{p1}\ge 20,\ \mathrm{peak\_balance}\ge 0.08,\ \mathrm{valley\_drop}\ge 0.08,\ \mathrm{minor\_drop}\ge 0.005,
\]
并满足左右主导性容差（\texttt{dominance\_tol}=0.02）。定义
\[
\mathrm{has\_double}:=\mathbf 1\{t_{p1},t_{p2}\ \text{均存在}\}.
\]
这些前置条件本身就是第一层“防伪双峰”过滤：
\begin{itemize}
  \item 平滑 $\tilde f=\mathrm{smooth}_7(f)$ 用来压掉格点奇偶振荡、台阶锯齿和数值毛刺，避免把离散噪声认成峰。
  \item 相对阈值 $0.06\,\max\tilde f$ 用来排除尾部极小 bump；如果一个“第二峰”连主峰的 $6\%$ 都不到，它更像尾部毛刺而不是真正的次峰。
  \item 时间间隔 $t_{p2}-t_{p1}\ge 20$ 用来排除单个宽峰肩部的小褶皱；间隔太小时，通常只是同一 lobe 的局部起伏。
  \item 候选峰平衡度 $\mathrm{peak\_balance}\ge 0.08$ 防止极弱次峰；否则大峰旁边一个很小的凸起也会被误判成双峰。
  \item $\mathrm{valley\_drop}\ge 0.08$ 要求谷底相对高峰有足够下沉，避免“只有一个浅凹口”的假分裂。
  \item $\mathrm{minor\_drop}\ge 0.005$ 进一步要求谷底也必须低于弱峰本身，专门排除近乎平顶或轻微波纹的 pseudo-second-peak。
  \item 左右主导性容差要求每个被选中的峰都必须是各自那一侧的近主导峰，从而挡掉“第二峰其实只是第一峰肩部”的 shoulder artifact。
\end{itemize}
因此：只有先通过这一层，后面的 phase 判据才有资格讨论“clear-double 还是 weak-double”。

\paragraph{(2) one-target 的 phase 定义}
先定义
\[
\mathrm{sep}_{\rm peaks}:=\frac{|t_{p2}-t_{p1}|}{w_{1/2}^{(1)}+w_{1/2}^{(2)}},\quad
\mathrm{valley}/\max:=\frac{h_v}{\max(h_{p1},h_{p2})},\quad
\mathrm{peak\_balance}:=\frac{\min(h_{p1},h_{p2})}{\max(h_{p1},h_{p2})}.
\]
其中 $w_{1/2}^{(k)}$ 为第 $k$ 峰半高宽（由原始 $f$ 计算），$h_{p1},h_{p2},h_v$ 由平滑序列 $\tilde f$ 在峰/谷位置读取。
分类为
\[
\mathrm{phase}_{\rm OT}=
\begin{cases}
2,& \mathrm{has\_double}=1,\ \mathrm{sep}_{\rm peaks}\ge 1.0,\\
1,& \mathrm{has\_double}=1,\ \mathrm{sep}_{\rm peaks}< 1.0,\\
0,& \mathrm{has\_double}=0.
\end{cases}
\]
因此 one-target 的硬 phase 判据现在只保留两条：
\begin{itemize}
  \item $\mathrm{has\_double}=1$ 仍然是最基本前提；若前置峰探测没有找到两个可信峰，phase 就保持 0。
  \item $\mathrm{sep}_{\rm peaks}\ge 1.0$ 要求两峰中心距至少与两峰总半高宽同量级；它防的是“一个宽峰上只有轻微起伏”或“两个峰严重重叠却被肉眼错看成双峰”。
\end{itemize}
而 $\mathrm{valley}/\max$ 与 $\mathrm{peak\_balance}$ 仍然保留并输出，但统一降级为 \emph{secondary diagnostics}：它们只负责解释双峰有多“干净”或多不对称，不再额外决定 phase。

\paragraph{(3) two-target 的 phase 定义}
先定义分量模态时间与分离度
\[
t_{\rm mode}^{\rm near}:=\arg\max_t f_{\rm near}(t),\quad
t_{\rm mode}^{\rm far}:=\arg\max_t f_{\rm far}(t),
\]
\[
\mathrm{sep}_{\rm mode}:=\frac{|t_{\rm mode}^{\rm far}-t_{\rm mode}^{\rm near}|}{w_{1/2}^{\rm far}+w_{1/2}^{\rm near}}.
\]
质量占比由分量总质量归一化得到
\[
P_{\rm near}:=\frac{\sum_t f_{\rm near}(t)}{\sum_t f_{\rm any}(t)},\qquad
P_{\rm far}:=\frac{\sum_t f_{\rm far}(t)}{\sum_t f_{\rm any}(t)}.
\]
并在总分布上定义
\[
\mathrm{peak\_ratio}:=\frac{\min(h_{p1},h_{p2})}{\max(h_{p1},h_{p2})},\qquad
\mathrm{valley}/\max:=\frac{h_v}{\max(h_{p1},h_{p2})}.
\]
分类为
\[
\mathrm{phase}_{\rm TT}=
\begin{cases}
2,& \mathrm{has\_double}=1,\ \mathrm{sep}_{\rm mode}\ge 1.0,\\
1,& \mathrm{has\_double}=1,\ \mathrm{sep}_{\rm mode}< 1.0,\\
0,& \mathrm{has\_double}=0.
\end{cases}
\]
two-target 情形现在也只保留最小的 hard gate：
\begin{itemize}
  \item $\mathrm{has\_double}=1$ 仍然是总曲线层面的必要条件；若总 FPT 根本没有两峰外观，phase 就保持 0。
  \item $\mathrm{sep}_{\rm mode}\ge 1.0$ 比较的是 $f_{\rm near}$ 与 $f_{\rm far}$ 的条件模态时间和宽度，而不是只看总曲线峰位；这就是 phase=2 唯一的硬分离门槛。
\end{itemize}
与此同时，$P_{\rm near},P_{\rm far}$、$\mathrm{valley}/\max$、$\mathrm{peak\_ratio}$ 等量继续输出，因为它们仍然适合做分支平衡与总曲线洁净度的诊断。但从数学定义上说，它们不再参与 phase gate：\textbf{phase=1} 现在只表示“已经检测到双峰，但分离度还不够”，而 \textbf{phase=2} 则表示“已经检测到双峰，且 near/far 模态分离足够明显”。

\subsection*{自适应 horizon guard 与 loss-mode 规则}
对 Track B 的 continuation 扫描，我们不允许把“second peak 没看到”直接当成“second peak 消失”。默认从 $t_{\max}=12000$ 开始，只要下面两条里有任何一条不满足，就把 horizon 加倍：
\[
\mathrm{survival\_tail}\le 10^{-12},
\qquad
t_{\mathrm{mode,far}} < 0.85\, t_{\max}.
\]
最大只加到 $t_{\max}=48000$；若到上限后仍不满足，就把该点标成 \texttt{tail\_truncation\_risk}。随后再按顺序应用 loss-mode 分类：
\[
\texttt{clear}\ \rightarrow\ \texttt{far\_mass\_loss}\ \rightarrow\ \texttt{near\_mass\_loss}\ \rightarrow\ \texttt{timescale\_merge}\ \rightarrow\ \texttt{far\_broadening}\ \rightarrow\ \texttt{tail\_truncation\_risk}.
\]
六个标签在本文中的具体定义是：
\begin{itemize}
  \item \texttt{clear}：该点仍满足 two-target 的全部 clear-double 条件，也就是 phase=2。
  \item \texttt{far\_mass\_loss}：$P_{\rm far}<0.15$，说明慢 far 分支的总质量先失重到不足以支撑第二峰。
  \item \texttt{near\_mass\_loss}：$P_{\rm near}<0.15$，说明快 near 分支先失重；这会让 early peak 塌掉，从而失去 clear-double。
  \item \texttt{timescale\_merge}：虽然两支质量都还在，但 $\mathrm{sep}_{\rm mode}<1.0$ 或 $t_{\rm mode}^{\rm far}-t_{\rm mode}^{\rm near}<60$，也就是 near/far 两条条件时间尺度已经并到一起，valley 不再清晰。
  \item \texttt{far\_broadening}：前面几条都没触发，但 far 分支已经宽到把慢峰抹平；可以理解为“第二峰还在尾部质量里，但不再形成清晰峰顶”。
  \item \texttt{tail\_truncation\_risk}：直到 $t_{\max}=48000$ 仍未通过 horizon guard，此时不能把“看不到第二峰”解释成真实物理失效，而必须先视为时域截断风险。
\end{itemize}
这样做的目的，是先判断“哪一支分支失重/变宽/并峰”，再判断是否属于真正的分类失效。本轮 start-scan 与 near-target-scan 中没有剩余的 \texttt{tail\_truncation\_risk}，因此这里看到的主导失效模式确实是物理性的 \texttt{near\_mass\_loss}，而不是时域截断假象；同时在当前 $(d,\Delta y)$ 扫描窗口里，实际只激活了 \texttt{clear}/\texttt{near\_mass\_loss}/\texttt{far\_broadening} 三类，所以相图上只会出现三种色块，其余颜色只是保留统一分类字典的位置。

\subsection*{committor 的数学机理（详细版）}
记 hitting time 为 $\tau_C:=\inf\{t\ge 0:\,X_t\in C\}$。在离散马尔可夫链转移矩阵 $P$ 上，设 $A$ 为起点盆（本报告用起点附近 strip），$B$ 为目标吸收集合，定义
\[
q_i := \mathbb P_i(\tau_B < \tau_A).
\]
它满足离散 Dirichlet 边值问题
\[
q_i=0\ (i\in A),\qquad q_i=1\ (i\in B),\qquad
q_i=\sum_j P_{ij} q_j\ (i\notin A\cup B).
\]
令 $U=\Omega\setminus(A\cup B)$，按 $(U,A,B)$ 重排后可写成线性系统
\[
(I-P_{UU})\,q_U=P_{UB}\,\mathbf 1.
\]
在本问题中（存在吸收且从 $U$ 终将离开）该系统解唯一，因此 $q$ 不是经验坐标，而是由动力学唯一确定的最优进度坐标。我们取 $q^*=0.5$ 定义承诺面
\[
\Sigma_{q^*}:=\{i:\ q_i\ge q^*\}.
\]
此外，在谱半径 $\rho(P_{UU})<1$ 时有 Neumann 展开
\[
q_U=(I-P_{UU})^{-1}P_{UB}\mathbf 1
=\sum_{n\ge 0}P_{UU}^n P_{UB}\mathbf 1.
\]
这意味着 $q_U$ 可以解释为“在内部区反复游走 $n$ 步后吸收到目标”的总和，从线性系统角度直接对应到后文的 renewal 失败-重试图景：早峰偏向小 $n$，慢峰来自大 $n$ 的累计权重。

\subsection*{L×R 事件定义与 recross 计数}
对轨迹 $X_t$（直到 $\tau_B$）定义膜泄漏次数
\[
N_{\rm mem}:=\sum_{t=0}^{\tau_B-1}\mathbf 1\!\{(X_t,X_{t+1})\in\mathcal E_{\rm mem}\},
\]
以及承诺后回撤次数
\[
N_{\rm ret}:=\sum_{t=0}^{\tau_B-1}\mathbf 1\!\{X_t\in\Sigma_{q^*},\,X_{t+1}\in A\}.
\]
二值分类变量取
\[
L=\mathbf 1\{N_{\rm mem}\ge 1\},\qquad R=\mathbf 1\{N_{\rm ret}\ge 1\},
\]
从而得到四类
\[
(\!L,R\!)\in\{(0,0),(0,1),(1,0),(1,1)\},
\]
对应 \texttt{L0R0/L0R1/L1R0/L1R1}（图~\ref{fig:membrane-class-legend}）。

\subsection*{与双峰/谷对应的 renewal 展开}
将首达分布记为 $f(t)$。若把“一次尝试直接成功”记为核 $g(t)$，把“先失败并回到再生集合后继续”记为核 $h(t)$，则
\[
f(t)=g(t)+(h*f)(t)=\sum_{k\ge 0}(h^{*k}*g)(t).
\]
其中 $k$ 可解释为失败尝试次数。双峰出现的条件就是 $k=0$ 与 $k\ge1$ 分量在时间尺度和质量上都可分离。窗口占比
\[
w_{\ell r}(W)=\frac{\sum_{t\in W} f_{\ell r}(t)}{\sum_{t\in W} f(t)}
\]
给出可视化证据：peak1 主要由 \texttt{L0R0} 托起；valley 仍以 \texttt{L0R0} 为主、但 \texttt{L1R0} 明显上升；peak2 则以 \texttt{L0R1} 主导，\texttt{L1R1} 仅占次要比例（图~\ref{fig:membrane-class-bars}，表~\ref{tab:membrane-window-classes}）。

\subsection*{从线性系统到首达核的闭环公式}
在无 corridor 双 target 场景，把非吸收内部态记为 $U$，起点内部基向量记为 $e_s$，并记
\[
Q:=P_{UU},\qquad R_{\rm near}:=P_{U,\partial_{\rm near}},\qquad R_{\rm far}:=P_{U,\partial_{\rm far}}.
\]
则首达 near/far 的 pmf 可写为
\[
f_{\rm near}(t)=e_s^\top Q^{t-1}R_{\rm near}\mathbf 1,\qquad
f_{\rm far}(t)=e_s^\top Q^{t-1}R_{\rm far}\mathbf 1,\qquad t\ge 1.
\]
相应总质量为
\[
P_{\rm far}=\sum_{t\ge 1}f_{\rm far}(t)=e_s^\top(I-Q)^{-1}R_{\rm far}\mathbf 1.
\]
这条公式把“时域首达曲线分解”与“线性系统解”直接连通，是后续 splitting committor 闭环的核心。

\subsection*{条件化核（Doob 变换）与峰谷转移解释}
在内部态且 $q_{\rm far}(i)>0$ 时，定义
\[
\widetilde P^{\rm far}_{ij}:=\frac{P_{ij}\,q_{\rm far}(j)}{q_{\rm far}(i)}.
\]
这就是以 $h=q_{\rm far}$ 对原转移核做的 Doob $h$-transform，对应“条件在最终先到 far 再到 near”下的精确动力学；near 分支则由互补的 $q_{\rm near}=1-q_{\rm far}$ 给出。于是 peak1 到 peak2 的窗口转移可解释为两类条件化轨道测度的再分配：早期质量主要来自短程成功轨道，而晚期质量来自多次离开并回到低-$q_{\rm far}$ 区域后的最终吸收。这一视角把观测到的 renewal 分解与可计算的条件化动力学直接对应起来。

\subsection*{谱分解视角：快慢双时标从何而来}
在内部态子核 $Q$ 上，若写成（广义）谱展开
\[
Q^{t-1}=\sum_{k}\lambda_k^{\,t-1} r_k\ell_k^\top,\qquad |\lambda_1|>|\lambda_2|\ge\cdots,
\]
则
\[
f_{\rm near}(t)=\sum_k c_k^{\rm near}\lambda_k^{\,t-1},\qquad
f_{\rm far}(t)=\sum_k c_k^{\rm far}\lambda_k^{\,t-1}.
\]
较小模值模态在早期快速衰减，主模 $\lambda_1$ 控制慢尾和晚峰。当 near/far 两分量在模态投影 $c_k^{\rm near},c_k^{\rm far}$ 上权重不同，就会出现“早期 near 主导、晚期 far 主导”的窗口迁移。这与 renewal 分解一致：peak1 对应短停留（小循环）贡献，peak2 对应准稳态主模累积释放。等价地看条件风险率
\[
h_{\rm far}(t):=\frac{f_{\rm far}(t)}{S(t-1)},
\]
其随时间上升就意味着远端吸收通道在晚窗口逐步占优。

\subsection*{双 target splitting committor 闭环}
对同一系统，定义
\[
q_{\rm far}(i):=\mathbb P_i(\tau_{\rm far}<\tau_{\rm near}),\qquad q_{\rm near}(i)=1-q_{\rm far}(i).
\]
其边值问题为
\[
q_{\rm far}=0\ \text{on near},\qquad q_{\rm far}=1\ \text{on far},\qquad
q_{\rm far}(i)=\sum_j P_{ij}\,q_{\rm far}(j)\ \text{on interior}.
\]
因此一级分解 $f=f_{\rm near}+f_{\rm far}$ 与 committor 可直接闭环：$q_{\rm far}(x_s)$ 应与 $P_{\rm far}$ 一致。定义
\[
\varepsilon_{\rm split}:=\left|q_{\rm far}(x_s)-\sum_{t\ge 1}f_{\rm far}(t)\right|
=\left|q_{\rm far}(x_s)-P_{\rm far}\right|.
\]
进一步地，令 $\tau:=\tau_{\rm near}\wedge\tau_{\rm far}$，并定义
\[
M_t:=q_{\rm far}(X_{t\wedge\tau}).
\]
由于 $q_{\rm far}$ 在 interior 上满足调和关系、在 near/far 边界上取 $0/1$，$M_t$ 是有界鞅。由可选停时定理可得
\[
q_{\rm far}(x_s)=\mathbb E[M_\tau]
=\mathbb P_{x_s}(\tau_{\rm far}<\tau_{\rm near})
=P_{\rm far}.
\]
因此 $\varepsilon_{\rm split}$ 理论上应为 $0$；数值实现中它的量级应与 Bellman 残差处于同一误差层级，可直接作为机制闭环与数值稳定性的联合质检量。

表~\ref{tab:tt-splitting-qc} 显示，physics anchor 与 clear-instance 两条路径都满足 $q_{\rm far}(x_s)\approx P_{\rm far}$，而且 $\varepsilon_{\rm split}$ 两行都已经到 $10^{-9}$ 量级，内点 Bellman 残差保持在 $10^{-11}$ 量级。说明“快/慢峰来自 near/far 吸收竞争”的解释在两条路径上都与精确分解一致。后续迭代中把 $\varepsilon_{\rm split}$ 与 committor 在内点的 Bellman 残差、near/far 边界残差一起作为自动质检量，可持续监控图表解释与数值分解是否漂移。

\subsection*{本报告中的 committor 实现细节}
\begin{enumerate}
  \item 在各个锚点案例上先构建精确转移矩阵（已包含懒惰项、偏置、膜穿透率和反射边界）。
  \item 取 $A=\{x\le x_s+1,\ |y-y_s|\le 1\}$，$B=\{\text{target}\}$，并用固定点迭代求解 $q$（迭代终止阈值 $10^{-11}$）；同时输出内点 Bellman 残差与边界残差作为数值质检，并在图~\ref{fig:one-target-basin} 中把 basin/承诺面画出来。
  \item 对 L×R 采用扩展状态空间 $(x,\ell,s,r)$ 的\textbf{精确递推}而非 MC：其中 $\ell$ 记录是否发生膜泄漏，$s$ 记录是否访问过 $\Sigma_{q^*}$，$r$ 记录是否在访问 $\Sigma_{q^*}$ 后返回 $A$。
  \item 默认报告 $q^*=0.5$，并额外输出 $q^*\in\{0.4,0.5,0.6\}$ 的鲁棒性表（表~\ref{tab:membrane-qstar-sens}）。
  \item 在双 target 无 corridor 场景中，一级分解始终采用严格恒等式 $f=f_{\rm near}+f_{\rm far}$，并额外输出完整的 $q_{\rm far}$ 场与 $q_{\rm far}=0.5$ 的位面，保证“双峰来源”可直接追溯。
\end{enumerate}

\section{总结与后续优化方向（committor 导向）}
\subsection*{实现总结（本轮优化后）}
\begin{enumerate}
  \item corridor 半透膜：对称/方向不对称代表案例均保持 clear-double；在严格 recross 定义下，peak1 主要是 \texttt{L0R0}，而慢峰由 \texttt{L0R1} 抬升，valley 中 \texttt{L1R0} 比例明显增加；新的 exact 空间占据图与方向通量图进一步把峰1/峰2 对应的区域与穿越方向直接画出来。
  \item 无 corridor 双 target：Track B 现在分成 physics anchor $(0.12,2,2)$ 与 showcase clear-instance $(0.12,4,1)$ 两条线。physics anchor 给出 $P_{\rm near}/P_{\rm far}\approx0.324/0.676$ 与 $(t_{p1},t_v,t_{p2})=(7,196,469)$；showcase clear-instance 则继续负责最清晰的图像，并通过首道 gate（\texttt{peak>=0.07}、\texttt{min>=0.07}、\texttt{valley<=0.10}）。同时新增输出 start-scan continuation CSV、$(d,\Delta y)$ loss-mode CSV，以及相应的 atlas/committor/loss-mode 图组。
  \item continuation 结果已经把两个物理问题具体化：\texttt{source\_only\_x} 扫描里 11 个点只有 4 个保持 \texttt{clear}，而控制组 \texttt{source\_plus\_near\_x} 的 11 个点全部保持 \texttt{clear}；$(d,\Delta y)$ 扫描中主导失效模式是 \texttt{near\_mass\_loss}（共 22 个），且在自适应 horizon guard 之后不再有 truncation-risk 假阳性。
  \item 方法层面：committor 从“术语”落到了可计算、可复核流程；splitting committor 在 physics anchor 与 clear-instance 上都与吸收质量形成数值闭环，并把 $\varepsilon_{\rm split}$ 与 committor 残差共同纳入自动质检（当前两行 $\varepsilon_{\rm split}$ 都在 $\sim10^{-9}$ 量级）。
  \item 工程执行层面：本轮本地完整回归（codegen + 中英文构建）单轮耗时约 $70\,$s（约 1.2 分钟），满足“本地 $<10$ 分钟优先”条件；后续仅在扫描规模扩大到预计单轮 $>20$ 分钟时切换 Isambard。
\end{enumerate}

\subsection*{后续方向（全部围绕 committor 落地）}
\begin{enumerate}
  \item 把当前已经落地的 physics-anchor continuation，从 1D 起点扫描和两个 $b_x$ 切片继续扩展到完整的 $(d,\Delta y,b_x)$ continuation 面，同时保留同一套 loss-mode bookkeeping。
  \item 做 $q^*$ 鲁棒性扫描（如 $0.4/0.5/0.6$）并报告峰谷分类稳定性，避免阈值依赖质疑。
  \item 把已经导出的 splitting committor 场与窗口统计进一步组织成整张 two-target 扫描上的阈值裕量图和路径家族 atlas
  \[
  q_{\rm far}(x)=\mathbb P_x(\tau_{\rm far}<\tau_{\rm near}),
  \]
  并继续把 $\varepsilon_{\rm split}=|q_{\rm far}(x_s)-P_{\rm far}|$ 与两项阈值裕量指标（\texttt{min-margin} 与 \texttt{gate-slack}）共同作为自动质检项。
  \item 在膜模型中继续扩展到“分段异质膜”（例如左/右不同 patch），并把它与当前方向不对称模型统一投影到 committor 几何上比较。
  \item 在 keepalive 循环中把“codegen + 中英文构建可通过”作为一等优化目标，并固定执行分流策略：本地小轮次（$<10$ 分钟）优先直跑；大规模扫描（预计单轮 $>20$ 分钟）走 isbard doctor/auth + submit/status/fetch；远端失败时先记录原因并回退最小本地 smoke。
  \item 所有远端轮次统一输出运行元信息（\texttt{JOB\_ID}、\texttt{REMOTE\_DIR}、关键产物路径与下一步动作），保证总结可追溯。
\end{enumerate}
