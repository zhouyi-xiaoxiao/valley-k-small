\section*{Headline Results}
This report adds two new research tracks on the 2D rectangular setting while keeping the same phase criteria and peak-valley style used in earlier reports.
\begin{enumerate}
  \item \textbf{Corridor + one target:} replace fully reflecting corridor walls by semi-permeable membranes, with both symmetric ($\kappa_\uparrow=\kappa_\downarrow$) and asymmetric ($\kappa_\uparrow\neq\kappa_\downarrow$) cases.
  \item \textbf{No corridor + two targets:} place one target close to the start and keep the other at the far side, to test robust ``fast near-hit + slow far-hit'' splitting.
\end{enumerate}

Main outcomes:
\begin{itemize}
  \item For symmetric membranes, clear-double forms a continuous ultra-low leakage band (see Figure~\ref{fig:sym-phase} and Table~\ref{tab:sym-w16}); it then enters a weak-double transition band and finally turns mostly single-peak at higher leakage.
  \item A clear asymmetric representative ($\kappa_\uparrow=0.002,\kappa_\downarrow=0$) keeps robust bimodality and changes valley/second-peak composition in the L$\times$R class decomposition.
  \item In the no-corridor two-target setup, near-start placement yields stable clear-double: the representative case $(b_x,d,\Delta y)=(0.07,2,3)$ gives near/far mass $\approx0.326/0.674$ with peaks around $(11,739)$; the dedicated showcase clear-instance is $(0.12,4,1)$ with near/far mass $\approx0.350/0.650$ and a deeper valley (valley/max $\approx0.007$).
\end{itemize}

\]

\subsection{Membrane Definition}
Each wall-crossing attempt at a membrane edge is accepted with probability $\kappa$ and reflected with probability $1-\kappa$.
Symmetric case: $\kappa_\uparrow=\kappa_\downarrow=\kappa$.
Asymmetric case: $\kappa_\uparrow\neq\kappa_\downarrow$.

\subsection{Class Labels for Mechanism}
We keep peak/valley windows and use a compact 4-class coding:
\begin{itemize}
  \item $L$: membrane leak happened or not;
  \item $R$: committor recross happened or not.
\end{itemize}
Classes are \texttt{L0R0}, \texttt{L0R1}, \texttt{L1R0}, \texttt{L1R1}.

\section{Method Reference (Committor + 2x2 Classes)}
\subsection*{Reproducible Definition of Phase and sep-score}
To make phase maps fully reproducible, we use the same criteria as in \path{src/vkcore/grid2d/rect_bimodality/cli.py}:
\paragraph{(1) Two-peak detection (\texttt{find\_two\_peaks})}
Let $\tilde f=\mathrm{smooth}_7(f)$ (window-7 smoothing). Candidate peaks satisfy
\[
\tilde f(t)>\tilde f(t-1),\quad \tilde f(t)\ge \tilde f(t+1),\quad \tilde f(t)\ge 0.06\,\max \tilde f.
\]
Pair selection then enforces (default code thresholds)
\[
t_{p2}-t_{p1}\ge 20,\ \mathrm{peak\_balance}\ge 0.08,\ \mathrm{valley\_drop}\ge 0.08,\ \mathrm{minor\_drop}\ge 0.005,
\]
plus side-dominance tolerance (\texttt{dominance\_tol}=0.02). Define
\[
\mathrm{has\_double}:=\mathbf 1\{t_{p1},t_{p2}\ \text{both exist}\}.
\]

\paragraph{(2) One-target phase definition}
Define
\[
\mathrm{sep}_{\rm peaks}:=\frac{|t_{p2}-t_{p1}|}{w_{1/2}^{(1)}+w_{1/2}^{(2)}},\quad
\mathrm{valley}/\max:=\frac{h_v}{\max(h_{p1},h_{p2})},\quad
\mathrm{peak\_balance}:=\frac{\min(h_{p1},h_{p2})}{\max(h_{p1},h_{p2})}.
\]
Here $w_{1/2}^{(k)}$ is half-width at half-maximum of peak $k$ (computed on raw $f$), while $h_{p1},h_{p2},h_v$ are read from smoothed $\tilde f$.
Then
\[
\mathrm{phase}_{\rm OT}=
\begin{cases}
2,& \mathrm{has\_double}=1,\ \mathrm{sep}_{\rm peaks}\ge 1.0,\ \mathrm{valley}/\max\le 0.35,\ \mathrm{peak\_balance}\ge 0.15,\\
1,& \mathrm{has\_double}=1\ \text{but the above clear-double test fails},\\
0,& \mathrm{has\_double}=0.
\end{cases}
\]

\paragraph{(3) Two-target phase definition}
Define modal times and mode-separation score
\[
t_{\rm mode}^{\rm near}:=\arg\max_t f_{\rm near}(t),\quad
t_{\rm mode}^{\rm far}:=\arg\max_t f_{\rm far}(t),
\]
\[
\mathrm{sep}_{\rm mode}:=\frac{|t_{\rm mode}^{\rm far}-t_{\rm mode}^{\rm near}|}{w_{1/2}^{\rm far}+w_{1/2}^{\rm near}}.
\]
Define normalized splitting masses
\[
P_{\rm near}:=\frac{\sum_t f_{\rm near}(t)}{\sum_t f_{\rm any}(t)},\qquad
P_{\rm far}:=\frac{\sum_t f_{\rm far}(t)}{\sum_t f_{\rm any}(t)}.
\]
On the total law, define
\[
\mathrm{peak\_ratio}:=\frac{\min(h_{p1},h_{p2})}{\max(h_{p1},h_{p2})},\qquad
\mathrm{valley}/\max:=\frac{h_v}{\max(h_{p1},h_{p2})}.
\]
Then
\[
\mathrm{phase}_{\rm TT}=
\begin{cases}
2,& \mathrm{has\_double}=1,\ \mathrm{sep}_{\rm mode}\ge 1.0,\ \min(P_{\rm near},P_{\rm far})\ge 0.15,\\
& \mathrm{valley}/\max\le 0.35,\ \mathrm{peak\_ratio}\ge 0.10,\ (\mathrm{peak\_ratio}-\mathrm{valley}/\max)\ge 0.01,\\
1,& \mathrm{has\_double}=1\ \text{but the above clear-double test fails},\\
0,& \mathrm{has\_double}=0.
\end{cases}
\]
So mathematically, \textbf{phase=1} means ``double peaks are detected, but at least one clear-double threshold is not met''.

\subsection*{Committor Mechanism (Detailed)}
Let the hitting time be $\tau_C:=\inf\{t\ge 0:\,X_t\in C\}$. On the discrete Markov chain with transition matrix $P$, let $A$ be the start basin (here a strip near the start) and $B$ be the absorbing target set. Define
\[
q_i := \mathbb P_i(\tau_B < \tau_A),
\]
which solves the discrete Dirichlet boundary-value problem
\[
q_i=0\ (i\in A),\qquad q_i=1\ (i\in B),\qquad
q_i=\sum_j P_{ij} q_j\ (i\notin A\cup B).
\]
With $U=\Omega\setminus(A\cup B)$, this is equivalent to the linear system
\[
(I-P_{UU})\,q_U=P_{UB}\,\mathbf 1.
\]
For this absorbing setup the solution is unique, so $q$ is a dynamics-determined progress coordinate (not an ad hoc geometric proxy).
We use $q$ as the progress coordinate and set $q^*=0.5$ to define the commitment surface
\[
\Sigma_{q^*}:=\{i:\ q_i\ge q^*\}.
\]
Moreover, when $\rho(P_{UU})<1$, the Neumann expansion gives
\[
q_U=(I-P_{UU})^{-1}P_{UB}\mathbf 1
=\sum_{n\ge 0}P_{UU}^n P_{UB}\mathbf 1.
\]
So $q_U$ is the cumulative contribution of paths that spend $n$ interior steps before absorption into $B$, which provides a direct linear-system bridge to the renewal failure-retry interpretation used later: early peak mass is small-$n$ dominated, while the slow peak accumulates from larger-$n$ loops.

\subsection*{L$\times$R Events and Recross Counts}
For a trajectory $X_t$ up to $\tau_B$, define membrane-cross count
\[
N_{\rm mem}:=\sum_{t=0}^{\tau_B-1}\mathbf 1\!\{(X_t,X_{t+1})\in\mathcal E_{\rm mem}\},
\]
and commitment-to-start recross count
\[
N_{\rm ret}:=\sum_{t=0}^{\tau_B-1}\mathbf 1\!\{X_t\in\Sigma_{q^*},\,X_{t+1}\in A\}.
\]
Then use binary labels
\[
L=\mathbf 1\{N_{\rm mem}\ge 1\},\qquad R=\mathbf 1\{N_{\rm ret}\ge 1\},
\]
which yields four classes
\[
(\!L,R\!)\in\{(0,0),(0,1),(1,0),(1,1)\},
\]
implemented as \texttt{L0R0/L0R1/L1R0/L1R1} (Figure~\ref{fig:membrane-class-legend}).

\subsection*{Peak/Valley Mechanism as Renewal Decomposition}
Let $f(t)$ be the FPT law. With direct-success kernel $g(t)$ and fail-return kernel $h(t)$,
\[
f(t)=g(t)+(h*f)(t)=\sum_{k\ge 0}(h^{*k}*g)(t).
\]
Here $k$ is the number of failed attempts before success. Bimodality appears when the $k=0$ (near one-shot) and $k\ge1$ parts separate in both timescale and mass. Window fractions
\[
w_{\ell r}(W)=\frac{\sum_{t\in W} f_{\ell r}(t)}{\sum_{t\in W} f(t)}
\]
make the transfer explicit: peak1 is mainly \texttt{L0R0}; valley remains \texttt{L0R0}-major with a clear rise of \texttt{L1R0}; and peak2 is \texttt{L0R1}-dominant, with \texttt{L1R1} secondary (Figure~\ref{fig:membrane-class-bars}, Table~\ref{tab:membrane-window-classes}).

\subsection*{Closed-Form Bridge from Linear System to FPT Kernels}
For the no-corridor two-target branch, let $U$ denote interior transient states, $e_s$ the interior basis vector at the start, and
\[
Q:=P_{UU},\qquad R_{\rm near}:=P_{U,\partial_{\rm near}},\qquad R_{\rm far}:=P_{U,\partial_{\rm far}}.
\]
Then the first-hit pmfs are
\[
f_{\rm near}(t)=e_s^\top Q^{t-1}R_{\rm near}\mathbf 1,\qquad
f_{\rm far}(t)=e_s^\top Q^{t-1}R_{\rm far}\mathbf 1,\qquad t\ge 1.
\]
Hence the far-hit mass is
\[
P_{\rm far}=\sum_{t\ge 1}f_{\rm far}(t)=e_s^\top(I-Q)^{-1}R_{\rm far}\mathbf 1.
\]
This gives the exact bridge between time-domain decomposition and the linear-system view used by committors.

\subsection*{Conditioned Kernel (Doob Transform) and Peak Transfer}
On interior states with $q_{\rm far}(i)>0$, define
\[
\widetilde P^{\rm far}_{ij}:=\frac{P_{ij}\,q_{\rm far}(j)}{q_{\rm far}(i)}.
\]
This is the Doob $h$-transform of $P$ by $h=q_{\rm far}$, i.e., the exact dynamics conditioned on eventually hitting \emph{far} before \emph{near}. In this view, the near branch is the complementary conditioned dynamics through $q_{\rm near}=1-q_{\rm far}$. Therefore, the window transfer from peak1 to peak2 can be interpreted as a measure transfer between two conditioned ensembles: early-time mass is dominated by short successful excursions, while late-time mass is amplified by loops that repeatedly leave and re-enter low-$q_{\rm far}$ regions before final absorption. This provides a direct dynamical meaning for the observed renewal split.

\subsection*{Spectral View: Origin of Fast/Slow Timescales}
On the interior kernel $Q$, write the (generalized) spectral expansion
\[
Q^{t-1}=\sum_k \lambda_k^{\,t-1} r_k\ell_k^\top,\qquad |\lambda_1|>|\lambda_2|\ge\cdots.
\]
Then
\[
f_{\rm near}(t)=\sum_k c_k^{\rm near}\lambda_k^{\,t-1},\qquad
f_{\rm far}(t)=\sum_k c_k^{\rm far}\lambda_k^{\,t-1}.
\]
Modes with smaller $|\lambda_k|$ decay quickly and dominate early windows, while the leading mode $\lambda_1$ controls the slow tail and late peak. If near/far branches project differently on these modes, we naturally obtain ``near-dominant early window'' versus ``far-dominant late window'' transfer. This is fully consistent with the renewal picture: peak1 is short-stay/low-loop mass, while peak2 is quasi-stationary release from the leading mode. Equivalently, the conditioned hazard
\[
h_{\rm far}(t):=\frac{f_{\rm far}(t)}{S(t-1)}
\]
increases over time when the far channel progressively takes over late windows.

\subsection*{Splitting Committor Closure for Two Targets}
On the same system, define
\[
q_{\rm far}(i):=\mathbb P_i(\tau_{\rm far}<\tau_{\rm near}),\qquad q_{\rm near}(i)=1-q_{\rm far}(i).
\]
Its boundary-value problem is
\[
q_{\rm far}=0\ \text{on near},\qquad q_{\rm far}=1\ \text{on far},\qquad
q_{\rm far}(i)=\sum_j P_{ij}\,q_{\rm far}(j)\ \text{on interior}.
\]
This gives a direct closure with the exact decomposition $f=f_{\rm near}+f_{\rm far}$: $q_{\rm far}(x_s)$ should match $P_{\rm far}$. Define
\[
\varepsilon_{\rm split}:=\left|q_{\rm far}(x_s)-\sum_{t\ge 1}f_{\rm far}(t)\right|
=\left|q_{\rm far}(x_s)-P_{\rm far}\right|.
\]
Moreover, let $\tau:=\tau_{\rm near}\wedge\tau_{\rm far}$ and define
\[
M_t:=q_{\rm far}(X_{t\wedge\tau}).
\]
Because $q_{\rm far}$ is harmonic on interior states and fixed to $0/1$ on near/far boundaries, $M_t$ is a bounded martingale. Optional stopping then gives
\[
q_{\rm far}(x_s)=\mathbb E[M_\tau]
=\mathbb P_{x_s}(\tau_{\rm far}<\tau_{\rm near})
=P_{\rm far}.
\]
So $\varepsilon_{\rm split}$ is theoretically zero; numerically, its scale should track the Bellman-residual scale, making it a joint diagnostic for mechanism closure and solver stability.

\subsection*{Committor Implementation in This Report}
\begin{enumerate}
  \item Build the exact transition kernel for each representative case (including laziness, drift, membrane permeability, and reflecting boundaries).
  \item Use $A=\{x\le x_s+1,\ |y-y_{\rm mid}|\le 1\}$ and $B=\{\text{target}\}$, and solve $q$ via fixed-point iteration with tolerance $10^{-11}$; record interior Bellman residual and boundary mismatch as quality diagnostics.
  \item Use exact augmented-state recursion on $(x,\ell,s,r)$ (not MC): $\ell$ tracks leak, $s$ tracks whether $\Sigma_{q^*}$ has been visited, and $r$ tracks strict recross (return to $A$ after visiting $\Sigma_{q^*}$).
  \item Report $q^*=0.5$ as default and additionally provide $q^*\in\{0.4,0.5,0.6\}$ robustness results (Table~\ref{tab:membrane-qstar-sens}).
  \item For no-corridor two-target cases, keep the strict identity $f=f_{\rm near}+f_{\rm far}$ as level-1 decomposition, then interpret windows with committor-consistent labels.
\end{enumerate}
